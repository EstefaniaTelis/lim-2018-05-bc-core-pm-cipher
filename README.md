# Cipher.me


![Pantalla](https://github.com/EstefaniaTelis/lim-2018-05-bc-core-pm-cipher/blob/gh-pages/pantalla-readme-cipher-me.png)

## Definición del producto

Cipher.me es una app web diseñada usando HTML, CSS y vanilaJS 

### Usuarios

Pensada principalmente para jovenes de 18 a 25 años, usuarios activos de la mensajería instantánea y las redes sociales, 
que poseen uno o más dispositivos móviles y quieren conservar la privacidad al enviar o recibir mensajes.

### Objetivos de los usuarios

Con Cipher.me los usuarios podrían cifrar y descifrar mensajes usando el Cifrado César. Al seleccionar una clave de cifrado y compartirla con sus amigos,
todos tendrán acceso exclusivo a desencriptar y encriptar todo el texto que necesiten para mantener sus conversaciones privadas.

##Diseño

El diseño está basado usando las últimas tendencias en diseño de interfaces, Material design. Con una paleta de colores moderna y 
basada en un color turquesa que brinda frescura y actualidad al look&feel.
